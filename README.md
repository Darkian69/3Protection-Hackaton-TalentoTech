# 3Protection-Hackaton-TalentoTech
Escaner de Seguridad TLS y Riesgo de Exposicion
Este proyecto consiste en una herramienta de visibilidad centralizada diseñada para auditar la seguridad de las comunicaciones web. El sistema analiza servidores, identifica configuraciones TLS inseguras, clasifica hallazgos segun su criticidad y genera reportes accionables para el endurecimiento de la seguridad (hardening).
